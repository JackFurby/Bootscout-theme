# Bootscout – WordPress theme for UK scout websites

![GitHub tag (with filter)](https://img.shields.io/github/v/tag/JackFurby/Bootscout-theme?label=Version)

---

**Bootscout is a free, open-source WordPress theme built for scouting websites in the UK.**

## Features

- On brand styling as detailed in the UK Scouts brand guidelines, including correct colours and fonts
- 5 built-in colour schemes (blue & yellow, green & navy, orange & forest green, red & pink, white & black)
- WordPress full site editing support (WordPress 6.8 or higher required)
- Scout-specific block patterns including section cards for all sections (Squirrels, Beavers, Cubs, Scouts, Explorers, Network)
- Multiple header and footer patterns
- Support for a number of popular WordPress plugins (support for other plugins can be requested by [opening an issue](https://github.com/JackFurby/Bootscout-theme/issues))

## Plugin support

**eCommerce**
- WooCommerce

**Forms**
- WPForms
- Fluent Forms

**Events**
- Simple Google Calendar Outlook Events Widget

**Locations**
- WP Store Locator

**Volunteering**
- Wired Impact Volunteer Management

**Utilities**
- Git Updater
- Quick and Easy FAQs
